1. Jalankan perintah npm install untuk memasang seluruh dependencies yang digunakan pada proyek.

2. Untuk menjalankan proyek, Anda bisa gunakan perintah-perintah berikut.

npm run start-dev: menjalankan proyek pada mode development menggunakan webpack-dev-server.
npm run build: membangun proyek dalam mode production.
